1. The Project must be run with Python3
2. The input file is given in "CarParking\Inputs\Sample_Input_File.txt, The action steps are mentioned here. This can be modifed to run steps for different scenario
3. The output is shown in the console
4. It can be run with separate Python env or in Python standalone
5. Once cloned or downloaded with zip and extracted, the default folder project name will be : "CarParkingProject-main"
6. To run the file do the following:
    i. set working directory as "..\CarParkingProject-main" : {Project location}\CarParkingProject-main
    ii. cd to ..\CarParkingProject-main : cd {Project location}\CarParkingProject-main
    iii. Run the command : python3 main_executor.py
